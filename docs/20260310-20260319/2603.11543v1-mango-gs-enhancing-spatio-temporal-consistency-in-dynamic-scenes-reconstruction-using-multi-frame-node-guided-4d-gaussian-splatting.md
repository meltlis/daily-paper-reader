## 摘要
重建具有照片级真实感细节和强时间相干性的动态 3D 场景仍然是一个重大挑战。现有的用于动态场景建模的高斯泼溅（Gaussian splatting）方法通常依赖于逐帧优化，这可能会过拟合到瞬时状态，而无法捕捉潜在的运动动力学。为解决这一问题，我们提出了 Mango-GS，这是一个用于高保真 4D 重建的多帧节点引导框架。Mango-GS 利用时间 Transformer 来建模短时间窗口内的运动依赖关系，从而产生时间一致的变形。为了提高效率，时间建模被限制在一组稀疏的控制节点上。每个节点由解耦的标准位置和潜码（latent code）表示，为运动传播提供稳定的语义锚点，并防止在大运动下的对应关系漂移。我们的框架采用端到端训练，并通过输入掩码策略和两个多帧损失函数进行增强，以提高鲁棒性。大量的实验表明，Mango-GS 达到了最先进的重建质量和实时渲染速度，实现了动态场景的高保真重建和交互式渲染。

## Abstract
Reconstructing dynamic 3D scenes with photorealistic detail and strong temporal coherence remains a significant challenge. Existing Gaussian splatting approaches for dynamic scene modeling often rely on per-frame optimization, which can overfit to instantaneous states instead of capturing underlying motion dynamics. To address this, we present Mango-GS, a multi-frame, node-guided framework for high-fidelity 4D reconstruction. Mango-GS leverages a temporal Transformer to model motion dependencies within a short window of frames, producing temporally consistent deformations. For efficiency, temporal modeling is confined to a sparse set of control nodes. Each node is represented by a decoupled canonical position and a latent code, providing a stable semantic anchor for motion propagation and preventing correspondence drift under large motion. Our framework is trained end-to-end, enhanced by an input masking strategy and two multi-frame losses to improve robustness. Extensive experiments demonstrate that Mango-GS achieves state-of-the-art reconstruction quality and real-time rendering speed, enabling high-fidelity reconstruction and interactive rendering of dynamic scenes.

动态 3D 场景重建旨在从视频中恢复具有照片级真实感且时间连续的场景。
*   **现有挑战**：目前基于 3D 高斯泼溅（3DGS）的动态建模方法大多采用“逐帧优化”策略，即独立预测每一帧的变形。这种方式容易导致模型“记住”瞬时状态而非学习运动规律，在面对快速或复杂运动时，往往会出现时间不一致、视觉伪影或模糊。
*   **核心动机**：为了增强时空一致性，需要引入多帧时序建模。但直接对数百万个高斯点应用 Transformer 会带来巨大的计算开销。因此，如何**在保持 3DGS 高效性的同时，引入稳健的多帧时序约束**是本文的核心问题。

### 2. 方法论：核心思想与关键技术
Mango-GS 提出了一种“稀疏引导密集”的多帧优化框架，核心流程如下：

*   **解耦控制节点表示（Decoupled Control Nodes）**：
    *   系统由一组稀疏的控制节点驱动。每个节点不再仅仅是一个 3D 位置，而是被解耦为**规范位置（Canonical Position）**和**潜码（Latent Code）**。
    *   这种设计通过学习“位置+特征”的联合亲和力来确定 k-NN 关系，有效防止了在大变形下，原本邻近的点因空间距离变化而产生的“邻域漂移”问题。
*   **多帧时间 Transformer（Temporal Attention Network）**：
    *   采用时间 Transformer 在一个短时间窗口（默认 $T=6$ 帧）内同时预测所有控制节点的运动轨迹。
    *   通过时间自注意力机制（Temporal Self-Attention）捕捉长程非线性依赖，生成物理上更合理的相干运动。
*   **运动传播与渲染**：
    *   将控制节点的变形（平移、旋转、缩放）通过预计算的权重传播给密集的高斯点云，最后利用标准 3DGS 渲染器进行实时渲染。
*   **训练策略与损失函数**：
    *   **时间输入掩码（Temporal Masking）**：随机遮盖部分时间嵌入，强迫网络利用上下文推理运动。
    *   **Top-k 硬帧损失（Top-k Hard-frame Loss）**：在每批次中仅针对误差最大的前 60% 帧计算光度损失，聚焦于挑战性时刻。
    *   **运动感知损失（Motion-aware Loss）**：通过计算相邻帧间的差异（幅度、方向、差值），显式监督时间相干性。

### 6. 主要结论与发现
*   **性能领先**：Mango-GS 在两个数据集上的 PSNR 和时间一致性指标（tLPIPS）均达到或超过了现有最先进水平。
*   **速度优势**：通过在稀疏节点空间进行时序建模，Mango-GS 成功保留了 3DGS 的实时渲染特性，速度比 MotionGS 和 TimeFormer 快 3 倍以上。
*   **鲁棒性**：解耦的节点表示显著提升了模型处理大范围、非刚性运动的能力，减少了伪影和 ghosting 现象。

### 7. 优点（亮点）
*   **高效的时序建模**：巧妙地将 Transformer 应用于稀疏节点而非全量高斯点，解决了计算开销与时序相干性之间的矛盾。
*   **语义稳定的邻域**：引入潜码（Latent Code）来辅助建立 k-NN 关系，比纯空间距离更具鲁棒性，有效应对了拓扑变化。
*   **训练策略精细**：Top-k 硬帧挖掘和运动感知损失的设计，针对性地解决了动态场景中“难点帧”和“闪烁”问题。

### 8. 不足与局限
*   **时间窗口限制**：目前采用固定长度的短窗口（$T=6$），虽然能捕捉局部相干性，但对于极长序列的全局一致性或超长程依赖可能仍有局限。
*   **极端运动挑战**：尽管引入了节点引导，但在面对极度剧烈、缺乏纹理或遮挡严重的运动时，仍可能依赖更强的几何先验或光流引导。
*   **内存开销**：虽然渲染快，但多帧 Transformer 在训练阶段的显存占用会随窗口大小增加，对显存容量有一定要求。
