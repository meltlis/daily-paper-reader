## 摘要
从稀疏、无位姿图像中进行语义感知的三维重建对于前馈式 3D 高斯泼溅（3DGS）而言仍然具有挑战性。现有方法在稀疏视图监督下通常会预测过完备的高斯基元集，导致几何结构不稳定且深度质量较差。同时，它们仅依赖 2D 分割器特征进行语义提升，这提供的 3D 层面监督较弱且泛化能力有限，导致在新场景中 3D 语义不完整。为了解决这些问题，我们提出了 UniSem，这是一个通过两个关键组件共同提高深度精度和语义泛化能力的统一框架。首先，误差感知高斯丢弃（EGD）通过利用渲染误差线索抑制易产生冗余的高斯分布，执行误差引导的容量控制，从而产生有意义且几何稳定的高斯表示，以改进深度估计。其次，我们引入了混合训练课程（MTC），该课程逐步将 2D 分割器提升的语义与模型自身涌现的 3D 语义先验相结合，并通过物体级原型对齐来实现，以增强语义的连贯性和完整性。在 ScanNet 和 Replica 上的大量实验表明，UniSem 在不同数量的输入视图下，在深度预测和开放词汇 3D 分割方面均取得了卓越的性能。值得注意的是，在 16 视图输入下，UniSem 相比强基准模型将深度相对误差（Rel）降低了 15.2%，并将开放词汇分割的平均准确率（mAcc）提高了 3.7%。

## Abstract
Semantic-aware 3D reconstruction from sparse, unposed images remains challenging for feed-forward 3D Gaussian Splatting (3DGS). Existing methods often predict an over-complete set of Gaussian primitives under sparse-view supervision, leading to unstable geometry and inferior depth quality. Meanwhile, they rely solely on 2D segmenter features for semantic lifting, which provides weak 3D-level and limited generalizable supervision, resulting in incomplete 3D semantics in novel scenes. To address these issues, we propose UniSem, a unified framework that jointly improves depth accuracy and semantic generalization via two key components. First, Error-aware Gaussian Dropout (EGD) performs error-guided capacity control by suppressing redundancy-prone Gaussians using rendering error cues, producing meaningful, geometrically stable Gaussian representations for improved depth estimation. Second, we introduce a Mix-training Curriculum (MTC) that progressively blends 2D segmenter-lifted semantics with the model's own emergent 3D semantic priors, implemented with object-level prototype alignment to enhance semantic coherence and completeness. Extensive experiments on ScanNet and Replica show that UniSem achieves superior performance in depth prediction and open-vocabulary 3D segmentation across varying numbers of input views. Notably, with 16-view inputs, UniSem reduces depth Rel by 15.2% and improves open-vocabulary segmentation mAcc by 3.7% over strong baselines.

#### 1. 核心问题与研究背景
*   **核心问题**：如何在仅有少量（稀疏）且没有相机位姿信息（unposed）的图像输入下，实现高质量、可泛化的三维几何重建与语义理解。
*   **研究背景**：
    *   现有的前馈式 3D 高斯泼溅（3DGS）方法在处理稀疏视图时，往往会为每个像素预测一个高斯基元，导致高斯点集过完备（冗余），从而在深度估计中引入大量噪声。
    *   现有的语义提升（Semantic Lifting）方法大多直接蒸馏 2D 分割模型（如 LSeg）的特征，这导致 3D 空间的一致性较弱，且受限于 2D 模型的泛化瓶颈，在新场景中语义表现不完整。

#### 2. 方法论
UniSem 提出了一个统一的框架，包含两个核心组件：
*   **误差感知高斯丢弃（Error-aware Gaussian Dropout, EGD）**：
    *   **核心思想**：在重建质量较好（低误差）的区域，冗余的高斯点对渲染贡献微小却会干扰梯度。EGD 通过计算每个像素的渲染误差，为高斯点分配丢弃概率。
    *   **技术细节**：误差越小的区域，高斯点被丢弃的概率越高。采用**余弦周期调度（Cosine-cycle Schedule）**来动态调整丢弃比例，避免训练初期不稳定。这种随机正则化强制模型生成更紧凑、几何更稳定的表示。
*   **混合训练课程（Mix-training Curriculum, MTC）**：
    *   **核心思想**：将 2D 分割器的监督与模型自身涌现的 3D 语义先验相结合。
    *   **技术细节**：
        1.  **最大误差提示（Max-error Prompting）**：在语义不一致最严重的像素点进行采样，利用 SAM2 模型获取跨视图的物体掩码。
        2.  **原型对齐（Prototype Alignment）**：在物体级别上，通过视图间原型一致性和几何感知原型对齐（利用点云密度加权），增强 3D 语义的连贯性和完整性。

#### 6. 主要结论与发现
*   **性能提升**：在 16 视图输入下，UniSem 相比强基准 Uni3R，深度相对误差（Rel）降低了 **15.2%**，语义分割平均准确率（mAcc）提升了 **3.7%**。

*   **场景局限性**：目前主要针对室内场景（ScanNet/Replica）进行训练和评估，在室外大场景、极端光照或动态场景下的表现尚未验证。

*   **掩码伪影风险**：在极度拥挤或杂乱的场景中，SAM2 生成的区域掩码可能存在伪影，从而影响语义对齐的质量。
