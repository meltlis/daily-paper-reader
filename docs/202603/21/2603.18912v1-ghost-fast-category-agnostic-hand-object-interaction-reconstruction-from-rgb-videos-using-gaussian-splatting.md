## 摘要
从单目 RGB 视频中理解真实的手-物交互对于 AR/VR、机器人技术和具身智能至关重要。现有方法依赖于特定类别的模板或繁重的计算，但在 3D 空间中产生的手-物对齐仍存在物理上的不一致性。我们提出了 GHOST（Gaussian Hand-Object Splatting），这是一个利用 2D 高斯泼溅重建动态手-物交互的快速、类别无关框架。GHOST 将手和物体都表示为稠密且视角一致的高斯圆盘，并引入了三项关键创新：(1) 几何先验检索和一致性损失，用于补全被遮挡的物体区域；(2) 抓取感知对齐，通过细化手部平移和物体尺度以确保真实的接触；(3) 手部感知背景损失，防止对被手部遮挡的物体区域进行惩罚。GHOST 能够从单段 RGB 视频中实现完整、物理一致且可动画化的重建，同时运行速度比之前的类别无关方法快一个数量级。在 ARCTIC、HO3D 和自然场景数据集上的广泛实验表明，该方法在 3D 重建和 2D 渲染质量方面达到了最先进的精度，确立了 GHOST 作为真实手-物交互建模的高效且鲁棒的解决方案。代码可在 https://github.com/ATAboukhadra/GHOST 获取。

## Abstract
Understanding realistic hand-object interactions from monocular RGB videos is essential for AR/VR, robotics, and embodied AI. Existing methods rely on category-specific templates or heavy computation, yet still produce physically inconsistent hand-object alignment in 3D. We introduce GHOST (Gaussian Hand-Object Splatting), a fast, category-agnostic framework for reconstructing dynamic hand-object interactions using 2D Gaussian Splatting. GHOST represents both hands and objects as dense, view-consistent Gaussian discs and introduces three key innovations: (1) a geometric-prior retrieval and consistency loss that completes occluded object regions, (2) a grasp-aware alignment that refines hand translations and object scale to ensure realistic contact, and (3) a hand-aware background loss that prevents penalizing hand-occluded object regions. GHOST achieves complete, physically consistent, and animatable reconstructions from a single RGB video while running an order of magnitude faster than prior category-agnostic methods. Extensive experiments on ARCTIC, HO3D, and in-the-wild datasets demonstrate state-of-the-art accuracy in 3D reconstruction and 2D rendering quality, establishing GHOST as an efficient and robust solution for realistic hand-object interaction modeling. Code is available at https://github.com/ATAboukhadra/GHOST.

#### 1. 核心问题与整体含义
在增强现实（AR/VR）、机器人和具身智能领域，从单目 RGB 视频中重建真实的手-物交互（HOI）至关重要。然而，现有方法面临三大挑战：
*   **类别限制**：许多方法依赖特定物体的 3D 模板，无法处理未知物体。
*   **物理不一致**：单目视角的深度歧义和严重遮挡常导致手部与物体接触不自然（如穿模或悬空）。
*   **计算效率低**：基于 NeRF 的方法通常需要数小时甚至十几小时的逐序列优化。
**GHOST** 旨在通过 **2D 高斯泼溅（Gaussian Splatting）** 技术，实现一个快速、无需物体类别先验、且物理接触一致的 3D 重建框架。

#### 2. 方法论
GHOST 框架分为三个主要阶段：
*   **预处理阶段**：
    *   使用 SAM2 进行物体分割，通过 SfM（如 HLoc 或 VGGSfM）估计相机位姿和稀疏点云。
    *   利用 HaMeR 初始化手部网格，并通过 VLM（如 InternVL）和 OpenShape 从 Objaverse 数据库中检索几何先验（3D 模型）。
*   **手-物对齐（HO Alignment）阶段**：
    *   **抓取检测**：通过比较手部和物体的运动轨迹相似度来识别抓取帧。
    *   **联合优化**：优化手部平移和物体尺度。引入**接触损失（Contact Loss）**，利用 Chamfer 距离强制手部与物体表面贴合；结合 2D 投影损失和时间平滑损失确保运动一致性。
*   **高斯泼溅优化阶段**：
    *   **物体重建**：引入**手部感知背景损失（Hand-aware Background Loss）**，避免在手部遮挡区域错误地剔除物体高斯点；使用**几何一致性损失（Geometric Consistency Loss）**，利用检索到的先验补全被遮挡的物体表面。
    *   **手部重建**：将高斯点绑定在 MANO 手部模型的面片上，通过仿射变换实现可动画化的手部渲染。

*   **对比方法**：主要对比了 **HOLD**（基于 NeRF 的类别无关方法）和 **BIGS**（基于高斯泼溅的最新方法）。

#### 6. 主要结论与发现
*   **速度优势**：GHOST 在保持甚至超越 SOTA 精度的情况下，大幅缩短了重建时间。
*   **接触质量**：通过抓取感知对齐，GHOST 产生的手-物接触比 HOLD 等方法更符合物理规律。
*   **遮挡补全**：引入几何先验能有效修复因手部遮挡导致的物体表面“空洞”问题。
*   **渲染性能**：在 ARCTIC 数据集上，其 PSNR 和 LPIPS 指标显著优于现有方法。

#### 7. 优点
*   **高效性**：将重建时间从“天/小时”级降低到“小时”级，更具实用潜力。
*   **类别无关**：不依赖预定义的物体模型库，通过在线检索实现通用性。
*   **物理一致性**：专门设计的接触损失解决了单目重建中常见的穿模问题。
*   **可动画化**：重建的手部模型可以直接通过参数驱动，适用于动画制作。

#### 8. 不足与局限
*   **SfM 依赖性**：重建质量高度依赖于预处理阶段 SfM 的位姿估计精度，若 SfM 失败，后续优化难以挽回。
*   **先验检索风险**：如果 VLM 识别错误或数据库中没有相似形状，检索到的几何先验可能误导重建（如补充错误的形状）。
*   **静态物体假设**：目前主要针对刚性物体，对于可变形物体（如海绵、布料）或具有复杂关节的物体处理能力有限。
*   **接触点固定限制**：如果手在物体上的接触点在整个视频中完全不改变，背景损失可能无法完全剔除手部区域的冗余高斯点。
